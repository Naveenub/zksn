(phase 2 re-randomizes these). alpha, beta, gamma are circuit-structure
constants and will be identical.

After running with pot28:
1. Export VK: `npx snarkjs zkey export verificationkey ceremony/zkey_final.zkey ceremony/verification_key.json`
2. Update `governance/contracts/Groth16Verifier.sol` — replace delta* and IC* constants
3. Regenerate proof: `node scripts/tree.js input ... | generate_witness | snarkjs prove`
4. Encode proof: `node scripts/encode_proof.js ceremony/proof.json`
5. Update `governance/test/ZKSNGovernance.t.sol` — REAL_PROOF and REAL_MEMBERSHIP_ROOT
6. `forge test` — all 47 tests must pass
7. Update this file with pot28 contribution hashes and SHA256s
